# CLAUDE.md - Project Guidelines for Claude Code

## Project Overview
This is a ROS2 robot project using {ros_distro} distribution.

## Coding Standards
- Use Python 3.8+ for scripts and nodes
- Follow ROS2 naming conventions
- Use rclpy for Python nodes
- Implement proper lifecycle management
- Add comprehensive logging

## Architecture Patterns
- Use composition over inheritance
- Implement proper error handling
- Use async/await for concurrent operations
- Follow the node lifecycle pattern

## Testing Requirements
- Unit tests for all utilities
- Integration tests for node interactions
- Launch file validation
- Parameter validation

## Documentation
- Add docstrings to all public functions
- Document launch file parameters
- Include usage examples

## Security Considerations
- Validate all input parameters
- Use secure parameter handling
- Avoid hardcoded credentials

## Performance Guidelines
- Minimize memory allocations in hot paths
- Use efficient data structures
- Profile performance-critical code
- Optimize message passing

## File Organization
- Keep related functionality together
- Use descriptive file names
- Organize by feature, not by type</content>
